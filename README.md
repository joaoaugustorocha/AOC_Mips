# Sistema de Alarme de Incêndio Inteligente Baseado em Cálculo de Risco em MIPS

Este trabalho apresenta o desenvolvimento do sistema de alarme de incêndio inteligente baseado em cálculo de risco em MIPS. O projeto foi desenvolvido em assembly e usa métricas de entradas de fumaça e temperatura para variar entre estados de risco, e caso necessário, ativar estado de emergência.
